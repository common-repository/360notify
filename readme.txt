== Description ==

### WooCommerce Order WhatsApp Notification ###

This is a plugin add-on for WooCommerce. But WooCommerce plugin is **not required** to access certain functionalities of our plugin. For example: Send WhatsApp.

Our WooCommerce Order WhatsApp Notification for Wordpress helps you to get your job done with minimal effort.

With our plugin, you can send WhatsApp Order Notifications to your customers (buyer) and administrators (seller/staff) automatically whenever a new order has been placed.

Try for FREE. After registration, use the service for free for 7 days.[360messenger.com](https://app.360messenger.com)

== Installation ==

1. Search for "360notify" in "Add Plugin" page and activate it.

2. Configure the settings in Settings > 360Notify WhatsApp Settings.

3. Enjoy.
